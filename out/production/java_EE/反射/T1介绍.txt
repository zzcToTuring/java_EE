1、什么是反射
    反射允许对成员变量，成员方法和构造方法的信息进行编程访问

在一个类中 --->存在字段(成员变量 Field)--->利用反射能获取字段--->
---->获取之后能：获取修饰符 获取名字 获取类型 赋值/获取值......

在一个类中--->存在构造方法（Constructor）--->利用反射能获取构造方法---->
---->获取之后能: 获取修饰符 获取名字  获取形参  创建对象

在一个类中--->存在成员方法(Method)--->利用反射能获取成员方法---->
---->获取之后能: 获取修饰符 获取名字  获取形参 获取返回值 创建对象 抛出异常 获取注解 运行方法

2、如何获取XX方法
首先我们要获取class对象，在根据不同的方法类型 获取不同的方法

3、反射的作用
    1、获取任意一个类中的所有信息
    2、结合配置文件动态创建对象

4、如何为java对象创建一个代理对象
    java.lang.reflect.Proxy类:提供了为对象产生代理对象的方法:
    public static Object newProxyInstance(ClassLoader loader, Class<?>11 interfaces, InvocationHandler h)
    参数一:用于指定用哪个类加载器，去加载生成的代理类
    参数二: 指定接口，这些接口用于指定生成的代理长什么，也就是有哪些方法
    参数三:用来指定生成的代理对象要干什么事情
    后直接把值返回即可
